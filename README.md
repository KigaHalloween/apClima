O projeto consiste em uma aplicação web interativa que permite aos usuários acessar informações climáticas em tempo real de qualquer lugar do mundo. 
Utilizando uma API climática confiável, como a OpenWeatherMap, o sistema permite que os usuários insiram o nome de uma cidade e obtenham dados detalhados
sobre a temperatura, umidade, velocidade. 
O intuito do projeto é o estudo de integração de API e desenvolvimento de uma interface amigável para o usuário.
Utilizado: JavaScript, HTML, Css e api “OpenWeatherMap”
